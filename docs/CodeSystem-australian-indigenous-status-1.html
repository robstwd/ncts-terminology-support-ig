CodeSystem: Australian Indigenous Status

The Australian Indigenous Status code system defines concepts to support the representation of a person’s Indigenous status. That is, whether a person identifies as being of Aboriginal or Torres Strait Islander origin, as represented by a code. The concepts defined in this code system are equivalent to those defined in the METeOR value domains for Person Indigenous status code N for http://meteor.aihw.gov.au/content/index.phtml/itemId/602545, http://meteor.aihw.gov.au/content/index.phtml/itemId/270885, and http://meteor.aihw.gov.au/content/index.phtml/itemId/291036.

This Code system is referenced in the content logical definition of the following value sets:

  • AustralianIndigenousStatus
